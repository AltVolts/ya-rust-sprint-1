# **Крейт `cli-converter`** (консольное приложение)


## Конвертер форматов записей банковских транзакций

Консольная утилита `cli-converter` использует функционал библиотеки `parser`.
Данная программа позволяет преобразовывать файлы с банковскими транзакциями из одного формата в другой. 
Для чтения и записи используется библиотека `parser`, построенная на трейтах `Read`/`Write`, 
поэтому конвертер может работать с любыми файлами, а результат выводить в стандартный поток вывода.


## Формат

### Структура утилиты
`cli-converter --input <input_file> --input-format <format> --output-format <format>`

### Аргументы
`--input <file>` (`-i`) — путь к входному файлу.

`--input-format <format>` (`-f`) — формат входного файла (`binary`, `csv`, `txt`).

`--output-format <format>` `-o`) — формат вывода (`binary`, `csv`, `txt`).

Результат конвертации выводится в `stdout`, поэтому его можно перенаправить в файл или обрабатывать дальше по конвейеру.

## Пример использования
```bash
cargo run -- --input ../test_files/records_example.csv --input-format csv --output-format txt > output_file.txt
```
